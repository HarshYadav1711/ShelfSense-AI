# ShelfSense AI Architecture (Scaffold)

- **Backend:** Django + Django REST Framework (`/api/v1`)
- **Frontend:** Next.js + TypeScript + Tailwind CSS
- **Metadata store:** MySQL
- **Vector store:** ChromaDB (local)
- **Scraper:** Selenium jobs under `scraper/`
- **Local LLM:** Ollama or LM Studio endpoint integration planned in `rag` app

This repository is intentionally scaffold-only for internship submission quality.
